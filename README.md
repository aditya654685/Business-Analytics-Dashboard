Sales Performance Case Study – Sales Dataset (Mantra Data Labs)

Key Highlights

Total Revenue: ₹39.9M | Units Sold: 7.6M | Revenue Growth: ~164%
Top Category: Yogurt (~₹8.23M); Top Brands: SnBrand2, YoBrand4
Regions (South/North/Central) and Channels (Retail/E-com/Discount) perform near-equally
Promotions show no clear uplift — A/B testing recommended

Tools: Excel 365, Power Query, PivotTables, Slicers
Workflow: Import & clean data → Engineer features (Revenue, Year, Month) → Build PivotTables/Charts per category → Assemble dashboard with KPI cards and connected slicers
